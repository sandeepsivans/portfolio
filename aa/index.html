What is HTML ?

HTML stands for Hyper Text Markup Language. it's a markup Language that web developers use to struture and describe the content if a webpage(not a programming Language)
HTML consists off element that describe types of content:paragraph, link, heading,images,video,etc.Web browser understand HTML code as websites.

In HTML,each elementis made up of 3 part:

  1. The opening tag
  2. The closing tag
  3. The actual element
